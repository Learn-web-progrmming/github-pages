مقدمة في Bootstrap

تعرف على أساسيات Bootstrap وكيفية إضافته لمشروعك. 

<!-- إضافة Bootstrap عبر CDN -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

<!-- أو التثبيت عبر npm -->
npm install bootstrap @popperjs/core

<!-- مثال بسيط -->
<button class="btn btn-primary">زر Bootstrap</button>

تخصيص Bootstrap

كيفية تخصيص Bootstrap ليتناسب مع احتياجات مشروعك. 

// ملف التخصيص (custom.scss)
$primary: #8b5cf6;
$secondary: #ec4899;

// إضافة متغيرات مخصصة
$custom-colors: (
  "custom-purple": #8b5cf6,
  "custom-pink": #ec4899
);

// دمج الخرائط
$theme-colors: map-merge($theme-colors, $custom-colors);

// استيراد Bootstrap
@import "~bootstrap/scss/bootstrap";

// أنماط مخصصة
.custom-class {
  background-color: $primary;
  color: white;
}

هذا مثال لتخصيص ملف Bootstrap، لا يوجد ناتج مرئي لهذا الدرس.

      <ul class="navbar-nav">
