المتغيرات

تعلم كيفية إنشاء واستخدام المتغيرات باستخدام let وconst. 

<script>
    let name = "أحمد";
    const age = 25;
    console.log(name, age);
</script>

أحمد 25 (يظهر في وحدة التحكم)

الدوال

إنشاء واستخدام الدوال لتنظيم الكود.

DOM Manipulation

الأحداث

إضافة تفاعلية باستخدام مستمعي الأحداث. 

<button id="myButton">اضغط هنا</button>

Promises

التعامل مع العمليات غير المتزامنة باستخدام Promises. 

<script>
    const myPromise = new Promise((resolve) => {
        setTimeout(() => resolve("تم التنفيذ!"), 1000);
    });
    myPromise.then(result => console.log(result));
</script>

تم التنفيذ! (يظهر في وحدة التحكم بعد ثانية)

Async/Await

تبسيط الكود غير المتزامن باستخدام async/await. 

<script>
    async function fetchData() {
        await new Promise(resolve => setTimeout(resolve, 1000));
        return "البيانات جاهزة!";
    }
    fetchData().then(data => console.log(data));
</script>

البيانات جاهزة! (يظهر في وحدة التحكم بعد ثانية)

أفضل الممارسات

كتابة كود JavaScript نظيف ومحسن. 

<script>
    const calculateSum = (a, b) => {
        if (typeof a !== "number" || typeof b !== "number") {
            throw new Error("المدخلات يجب أن تكون أرقام");
        }
        return a + b;
    };
    try {
        console.log(calculateSum(5, 10));
    } catch (error) {
        console.error(error.message);
    }
</script>
