مقدمة في HTML

تعرف على أساسيات HTML وكيفية إنشاء صفحة ويب بسيطة. 

<!DOCTYPE html>
<html lang="ar">
<head>
    <meta charset="UTF-8">
    <title>صفحتي الأولى</title>
</head>
<body>
    <h1>مرحبا بالعالم!</h1>
</body>
</html>

  • عنصر قائمة 2

الجداول

    </tr>

        <td>بيانات 2</td>
    </tr>
</table>

إمكانية الوصول

جعل الموقع متاحًا للجميع باستخدام ARIA.

<button aria-label="إغلاق النافذة">X</button>
